# django-fluo-posts #

A generic extensible django application to handle posts, as news or blog applications.
Based on `django-fluo`.


## Install ##

Add `posts.apps.PostsConfig` into your `INSTALLED_APPS`

```
#!python
INSTALLED_APPS = [
    ...
    "posts.apps.PostsConfig",
    ...
]
```
